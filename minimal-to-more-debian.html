Minimal to More Debian

  Last modified on Saturday 10 June 2017

[[ DRAFT ]]

Start with a minimal install of Debian 9.0 "Stretch" and roll out a customized home server, or a workstation using the lightweight i3 tiling window manager. Or whatever you like ... Debian is all about lots(!) of libre software choices!

Let's go!

Pick a machine

Maybe transform a chromebook into a cheap and cheerful full-featured Linux laptop?. Or perhaps dust off an old laptop and give it new life as a home server?

Download

Download the minimal network installer (official or unofficial non-free images including firmware packages). Verify the downloaded installer image file is OK by checking the PGP signature using GnuPG.

Installer

Prepare the installation media or supercharge that installer by making a GRUB multi-boot device!

Go minimal

A visual walk-through of a sample setup that includes encryption + the Logical Volume Manager (LVM) to create "virtual partitions" (Logical Volumes).
