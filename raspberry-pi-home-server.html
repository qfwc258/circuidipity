There's no place like [a Raspberry Pi] Home [Server]

  Last modified on Thursday 19 February 2015

Work-in-progress: Running your own Linux-powered home server is a fun learning experience about how networks work and using a Raspberry Pi is a cheap and cheerful way to get the job done!

Let's go!

0. Move the root filesystem to external USB storage

1. Secure remote access using SSH keys

2. Multiple terminal windows using tmux

3. Backup /home

4. Network attached storage

5. Access from anywhere in the world using dynamic DNS

6. Web server and database

7. RSS reader and aggregator

Happy hacking!
