Circuidipity is made possible thanks to the awesomesauce of free and open-source software. The original source of this website is composed on a laptop running Debian Linux using vim and reStructuredText then compiled by Pelican using Chungking Condo theme into static HTML files. Images are created and processed using GIMP. Fonts used are Yanone Kaffeesatz and Merriweather.

I use s3cmd to upload content to Amazon S3 for hosting. Espresso, bananas, serendipity and friends are my duct tape.
